# login
* 供使用者登入

# 功能描述 (features)
* 使用者提供帳號密碼，伺服器檢查身份

# 環境建置與需求 (prerequisites)
* Node.js: 13.5.0
* Express: 4.17.1
* Express-Handlebars: 5.2.0
* body-parser: 1.19.0
* express-session: 1.17.1

# 安裝與執行步驟(installation and execution)
  1. 打開終端機(Terminal)，Clone 此專案至本地電腦
  ```
  git clone https://github.com/yun856839/login.git
  ```

  2. 開啟終端機，進入專案資料夾
  ```
  cd user_login
  ```

  3. 安裝 npm 套件
  ```
  npm install
  ``` 

  4. 執行 server
  ```
  npm run dev
  ```

  5. 開啟任一瀏覽器瀏覽器，輸入網址
  ```
  http://localhost:3000
  ```
